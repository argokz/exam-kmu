# exam-kmu

Контрольная по Python: генерация задания и проверка через GPT.

- **exam_core.py** — служебный модуль (ключ API, генерация и проверка заданий). Подключается в ноутбуках по raw-URL: `https://raw.githubusercontent.com/argokz/exam-kmu/main/exam_core.py`. Код не хранится в ячейках — только загрузка из Git. Результат — оценка 0–100 и комментарий (описание ошибок при наличии).

## Как открыть в Google Colab

```
https://colab.research.google.com/github/argokz/exam-kmu/blob/main/Exam_KMU.ipynb
```

Либо: **File → Open notebook → GitHub** → `argokz/exam-kmu` → `Exam_KMU.ipynb`.

## Настройка (преподаватель, один раз)

1. В Colab откройте **Secrets** (иконка замка слева в панели).
2. Добавьте секрет с именем **`OPENAI_API_KEY`** и значением вашего ключа OpenAI.
3. Запустите **первую ячейку с кодом** (настройка) один раз. Студентам её не показывать — они работают начиная с ячейки **«ФИО, группа, кредит»**.

Ключи хранятся только в Colab Secrets, в репозитории их нет.

## Результаты

- В ноутбуке студент видит **оценку** и **комментарий** (в т.ч. описание ошибок). Записывайте результаты сами или используйте опцию ниже.
- **Опционально:** при первом запуске настройки можно смонтировать Google Drive; тогда результаты будут дописываться в файл **`/content/drive/MyDrive/exam_results.db`** (SQLite). Таблица: `fio`, `group_name`, `credit`, `variant`, `score`, `feedback`, `created_at`.

## Порядок для студента

1. Указать **ФИО**, **группу**, **кредит** и нажать **Сгенерировать задание**.
2. Написать решение в ячейке с `student_code` и запустить ячейку.
3. Нажать **Проверить** — отобразятся оценка и комментарий.
